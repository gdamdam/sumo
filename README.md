# Sumo 
a tool for semantic analysis of web articles.


## Main requirements
<pre>
MongoDB >=2.6.5 
Python >=2.7.5
</pre>

## Basic Installation

<pre>
git clone https://github.com/gdamdam/sumo.git
cd sumo
pip install -r requirements.txt
python requirements_nltk.py
</pre>

## Usage
Just lunch the server

<pre>
python sumo_server.py
</pre>
